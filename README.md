## Описание

В данном репозитории размещена статья по результатам производственной технологической практики на тему:

**«Разработка модуля размещения, поиска и управления объявлениями для web-ресурса АН „Город“»**

В статье рассматривается разработка модуля объявлений, предназначенного для автоматизации размещения, поиска и управления объявлениями на web-ресурсе организации. Разработанный модуль включает каталог объявлений, поиск и фильтрацию, подробную карточку объявления, публикацию, личный раздел пользователя, избранное, модерацию и обмен сообщениями между пользователями.

## Аннотация

Актуальность работы связана с необходимостью автоматизации процесса размещения и поиска объявлений. Ручная обработка объявлений занимает больше времени и не обеспечивает достаточного удобства для пользователей. Собственный web-модуль позволяет централизованно хранить данные, организовать удобный поиск, управлять публикациями и контролировать качество размещаемой информации.

В ходе работы были проанализированы существующие сервисы объявлений, сформулированы требования к разрабатываемому модулю, выбраны технологии разработки, спроектирована база данных и реализованы основные функции web-приложения.

## Ключевые слова

`web-приложение` `Django` `Python` `база данных` `объявления` `каталог` `модерация` `избранное` `личный кабинет`

## Используемые технологии

- **Python** — реализация серверной логики web-приложения;
- **Django** — маршрутизация, модели, формы, шаблоны и административная панель;
- **SQLite** — хранение данных пользователей, объявлений, фотографий, избранного и сообщений;
- **HTML, CSS, JavaScript** — структура страниц, оформление интерфейса и интерактивные элементы;
- **AJAX** — асинхронное взаимодействие с сервером без полной перезагрузки страницы.

## Основные функции модуля

- просмотр каталога объявлений;
- поиск и фильтрация по названию, категории, городу, цене и параметрам;
- просмотр подробной карточки объявления;
- публикация и редактирование объявлений;
- загрузка фотографий;
- автоматическое формирование заголовков;
- добавление объявлений в избранное;
- личный раздел пользователя;
- переписка между покупателем и продавцом;
- модерация объявлений;
- управление статусами публикаций;
- отправка сообщения об ошибке.

## Результат

В результате работы был разработан модуль размещения, поиска и управления объявлениями для web-ресурса АН «Город». Модуль обеспечивает полный цикл работы с объявлениями: от просмотра и поиска до публикации, редактирования, модерации, добавления в избранное и обмена сообщениями между пользователями.

Разработанный модуль может использоваться как основа web-ресурса АН «Город» и дорабатываться в рамках выпускной квалификационной работы.
